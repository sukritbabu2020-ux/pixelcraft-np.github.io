# PixelCraft Web Designer Portfolio

## Overview
A high-end, professional portfolio website balancing minimalist sophistication with vivid, bombastic graphics. Features:
- Stunning hero with parallax particle animation
- About section with creative process
- Interactive project submission portal with file preview
- Contact form
- Smooth animations, premium typography, responsive design

## Quick Start
1. Open `index.html` in your browser.
2. To make forms fully functional:
   - Sign up at [EmailJS.com](https://www.emailjs.com/)
   - Replace placeholders in `js/script.js`:
     - `YOUR_SERVICE_ID`
     - `YOUR_PROJECT_TEMPLATE_ID` (create template sending to creatorofwebsites57@gmail.com)
     - `YOUR_CONTACT_TEMPLATE_ID`
     - `YOUR_PUBLIC_KEY`
3. Add your logo/images to `assets/` and update paths.
4. Host on Netlify/Vercel for production.

## Customization
- **Vivid Images**: Replace Unsplash URLs with your high-res design mocks.
- **Animations**: Enhance with GSAP CDN if needed.
- **Typography**: Inter + Playfair Display via Google Fonts.

## Tech Stack
- Vanilla HTML/CSS/JS
- EmailJS for forms
- CSS Grid/Flexbox, Canvas for particles, Intersection Observer

Built for elite performance and "wow-factor" client conversions.
